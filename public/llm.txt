> Climate Resilience Assessment Platform for Healthcare Facilities

## Platform Overview

**Name**: Pathways Health
**Domain**: Climate-Resilient Healthcare | Healthcare Infrastructure Assessment | Climate Adaptation
**Purpose**: Evidence-based climate resilience assessment tool for healthcare facilities worldwide
**Target Audience**: Healthcare administrators, facility managers, climate resilience officers, public health officials, healthcare donors, international development organizations

## Core Value Proposition

Pathways Health is the first comprehensive, evidence-based assessment platform designed to help healthcare facilities measure and improve their resilience to climate change impacts. We provide actionable insights across critical infrastructure pillars to protect healthcare delivery during climate events.

## Key Features & Capabilities

### 1. Climate Resilience Assessment
- Multi-hazard vulnerability assessment (floods, storms, heatwaves, droughts, wildfires, sea level rise, cold waves)
- Four-pillar evaluation framework: Workforce Capacity, WASH & Waste Management, Energy Services, Infrastructure
- Evidence-based scoring system with weighted critical questions
- Real-time progress tracking and multi-section navigation
- Mobile-responsive assessment interface

### 2. Dashboard & Analytics
- Historical assessment tracking and comparison
- Per-assessment PDF report generation
- Pillar-specific resilience scores with visual indicators
- Comprehensive action plan generation
- Risk level categorization (High Resilience, Medium Risk, High Risk)

### 3. Interactive Reports
- Dynamic report preview with save, download, and share functionality
- Anonymous assessment support with claim functionality
- Email notification system for assessment summaries
- Cloud storage integration for report sharing
- Sign-in gating for premium features

### 4. Data-Driven Recommendations
- Hazard-specific vulnerability identification
- Priority-based action planning (Major/Medium priority)
- Pillar-specific mitigation strategies
- Impact statement mapping to infrastructure gaps
- Evidence-based intervention recommendations

## Technical Architecture

### Technology Stack
- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes, Supabase PostgreSQL
- **Authentication**: Clerk (secure user management)
- **Database**: Supabase (real-time capabilities, Row Level Security)
- **File Storage**: Google Cloud Storage
- **Email**: Resend (transactional emails)
- **Analytics**: Custom event tracking, Vercel Analytics
- **Monitoring**: Sentry (error tracking and performance)
- **PDF Generation**: @react-pdf/renderer

### API Endpoints
- `/api/assessment/submit` - Submit anonymous or authenticated assessments
- `/api/assessment/save` - Claim anonymous assessments post-authentication
- `/api/dashboard` - Retrieve user assessment history and analytics
- `/api/questions` - Fetch vulnerability questions by hazard type
- `/api/reports/upload` - Upload and share assessment reports
- `/api/analytics/track` - Track user events and engagement
- `/api/partner` - Partner organization submission endpoint

### Database Schema
**Core Tables**:
- `Assessment` - Assessment records with facilityName, location, userId, date, isAnonymous
- `VulnerabilityQuestion` - Evidence-based questions with hazard, pillar, weight, isCritical
- `ImpactStatement` - Climate impact descriptions by hazard and pillar
- `AssessmentResponse` - User responses to vulnerability questions
- `ImpactResponse` - Identified impacts with severity levels
- `AnalyticsEvent` - User engagement and behavior tracking

## Methodology & Approach

### Evidence Base
Our assessment framework is built on:
- WHO guidelines for climate-resilient healthcare
- IPCC climate impact assessments
- Global health security framework
- Infrastructure resilience standards (ASCE, ISO)
- Field research from climate-vulnerable regions

### Assessment Pillars

**1. Workforce Capacity**
- Staff training on climate emergencies
- Backup staffing protocols
- Health worker safety during climate events
- Telemedicine and remote care capabilities

**2. WASH & Waste Management**
- Water supply security and backup systems
- Sanitation continuity during disruptions
- Medical waste management in emergencies
- Infection prevention and control

**3. Energy Services**
- Backup power generation capacity
- Energy storage and distribution
- Critical medical equipment power requirements
- Renewable energy integration

**4. Infrastructure**
- Building structural integrity
- Flood protection and drainage
- Temperature control systems
- Emergency access and evacuation routes
